MYRIAD PRO — Web Font Files
===========================
Place your licensed Myriad Pro web font files in this folder.

Required files (woff2 + woff for each weight):

  MyriadPro-Light.woff2
  MyriadPro-Light.woff

  MyriadPro-Regular.woff2
  MyriadPro-Regular.woff

  MyriadPro-Semibold.woff2
  MyriadPro-Semibold.woff

  MyriadPro-Bold.woff2
  MyriadPro-Bold.woff

How to get web fonts from your Adobe/Typekit licence:
  1. Log in to fonts.adobe.com
  2. Find Myriad Pro in your active fonts
  3. Use a tool such as fontsquirrel.com/tools/webfont-generator
     to convert the desktop OTF/TTF files to woff2/woff
  4. Rename to the file names above and drop into this folder

Until the files are placed here, the site falls back to
Source Sans 3 (loaded from Google Fonts) which is visually
very close to Myriad Pro.
